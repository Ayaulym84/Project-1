Игра "Камень-Ножницы-Бумага"
Сделана в PyCharm
Суть игры: Вы играете против компьютера и Вам необходимо сделать выбор: камень, ножницы, бумага.
Правила: камень бьет ножницы, ножницы бьют бумагу, а бумага камень.
Игра будет продолжаться до тех пор, пока либо у Вас, либо у компьютера не наберется 3 очка.
